Citi Loyalty Program

This web application demonstrates a customer loyalty program on blockchain using Hyperledger API

Learn more »

Customer

Customers can register as Member to this program by creating an account number for identification on the network to sign-in. Once registered they can sign in, to make transactions to use points, and view all their transactions.

Sign In » Register »

Partners

For companies part of the program, they can register as Partner on the network by creating an Id on the network to sign-in. They can sign-in to view all transactions made with them and dashboard to view total points allocated and redeemed by customers.
